关于语言模型(LM)

发表于 2022-03-03 更新于 2022-03-05 分类于 知识整理

前言

在NLP的很多引用中，我们都需要判断一个句子是否合理，比如机器翻译中，需要判断翻译的话是否合理；拼写检查中，正确的词在语句中出现的概率要比错误的词出现的概率更高；在语音识别中，对于同音词的处理。

最初的语言模型是基于语法规则手工编织的，后来被统计语言模型取代。

在机器学习中，对上述需求的做法是计算概率，当一个句子出现的概率较大时，我们可以判断该句更为合理，更容易出现。根据大数定理，当样本量足够大时，概率可以由频率来进行估计，所以我们可以统计频率，再以此估计概率。

后续随着神经网络的发展，提出了神经网络语言模型，核心思想是神经网络的强大的拟合能力，直接拟合出语言模型的概率分布。

在本博客中，主要重点介绍统计语言模型和简单的神经网络语言模型(NNLM)。

关于TF-IDF

发表于 2022-02-28 分类于 知识整理

TF-IDF

TF-IDF(term frequency-inverse document frequency)用于信息检索和数据挖掘的常用加权技术. TF是词频, IDF是逆文本指数

用于评估一个字词对于一个文件集或一个语料库中的其中一份文档的重要程度, 用作搜索信息检索, 文本挖掘和用户建模的加权因子.

关于BM25

发表于 2022-02-26 更新于 2022-03-04 分类于 知识整理

BM25(Best Match)

BM25算法是在20世纪70年代到80年代由英国的一批信息检索领域的计算机科学家发明, 用来衡量搜索词query和文档document相似度得分的经典算法，这个相关性打分是一个类似TF-IDF的基于统计计数的无监督学习过程.
是一种基于词袋方法的词频计数, 它不考虑多个搜索词在文档中的关联性, 只考虑它们各自的出现次数
